# 🧠 一、题目在干嘛？

👉 给你一个数组：
```text
[2,3,1,2,4,3]
```

👉 还有一个目标：
```text
target = 7
```

👉 你要找：
👉 **一段连续的数字（子数组）**
👉 让它们的和 ≥ 7
👉 并且这段要 **最短**

# 🧩 二、什么是“子数组”？
👉 必须是**连续的！**
✅ 可以：

```text
[2,3,1]
[1,2,4]
[4,3]
```

❌ 不可以：
```text
[2,1,4]  ❌（不连续）
```

# 🪟 三、什么是“滑动窗口”？（重点🔥）
想象你有一个“框”👇
```text
nums = [2,3,1,2,4,3]

窗口：
[2,3]
```

👉 这个框可以：
* 向右移动 👉
* 变大
* 变小

# 🎯 核心思想（超重要）
👉 我们用一个“窗口”去**扫整个数组**

### 两个指针：
* `left` 👉 窗口左边
* `right` 👉 窗口右边

# 🔄 四、一步一步滑动（重点理解）
## 初始：
```text
left = 0, right = 0
窗口 = []
sum = 0
```

## 👉 Step 1：扩大窗口（右边动）
```text
right = 0 → 加入 2

窗口：[2]
sum = 2 ❌ (<7)
```

## 👉 Step 2：继续扩大
```text
right = 1 → 加入 3

窗口：[2,3]
sum = 5 ❌
```

## 👉 Step 3：
```text
right = 2 → 加入 1

窗口：[2,3,1]
sum = 6 ❌
```

## 👉 Step 4：
```text
right = 3 → 加入 2

窗口：[2,3,1,2]
sum = 8 ✅（满足）
```

# 🔥 五、关键来了：开始“缩小窗口”

👉 因为我们要**最短**
## 👉 缩小（left动）
```text
去掉 2

窗口：[3,1,2]
sum = 6 ❌
```

👉 不满足了 → 停止缩小
## 👉 Step 5：继续扩大
```text
right = 4 → 加入 4

窗口：[3,1,2,4]
sum = 10 ✅
```

## 👉 再缩小！
```text
去掉 3 → [1,2,4] sum=7 ✅
去掉 1 → [2,4] sum=6 ❌
```

👉 当前最短 = 3
## 👉 Step 6：

```text
right = 5 → 加入 3

窗口：[2,4,3]
sum = 9 ✅
```

## 👉 再缩！
```text
去掉 2 → [4,3] sum=7 ✅
👉 长度 = 2 ⭐（最短）
```

# 🏁 最终答案
```text
[4,3] → 长度 2
```

# 🧠 六、总结滑动规则（超级重要）

👉 就两个动作：
## ① 右边扩展（加数字）

```python
total += nums[right]
```

## ② 满足条件就缩左边（核心🔥）

```python
while total >= target:
    更新最小长度
    total -= nums[left]
    left += 1
```

# 💻 七、代码（对应上面步骤）

```python
def minSubArrayLen(target, nums):
    left = 0
    total = 0
    min_len = float('inf')

    for right in range(len(nums)):
        total += nums[right]  # 👉 扩大窗口

        while total >= target:  # 👉 满足就缩小
            min_len = min(min_len, right - left + 1)
            total -= nums[left]
            left += 1

    return 0 if min_len == float('inf') else min_len
```

# 🎯 八、一句话记住滑动窗口
**右边一直加，够了就左边减**

# 🚀 九、你现在要掌握的关键点

如果你只记3点👇

1. 👉 两个指针（left / right）
2. 👉 right 负责扩大
3. 👉 left 负责缩小（while！）

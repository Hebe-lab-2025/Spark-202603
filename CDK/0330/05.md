# 🤖 什么是 AWS CodeWhisperer？

👉 AWS 官方的 **AI 编程助手（类似 GitHub Copilot）**

📌 作用：

* 自动补全代码 ✨
* 生成 AWS CDK 代码 🚀
* 提供安全建议 🔐

---

# 🧠 它能帮你做什么？

## ✅ 1. 自动生成 CDK 代码

例如你写：

```ts
// create a lambda function
```

👉 CodeWhisperer 会自动补全：

```ts
new lambda.Function(this, 'MyFunction', {
  runtime: lambda.Runtime.NODEJS_18_X,
  handler: 'index.handler',
  code: lambda.Code.fromAsset('lambda'),
});
```

---

## ✅ 2. 自动写 UserData / Shell 脚本

```bash
# install nginx
```

👉 自动补全完整安装步骤

---

## ✅ 3. 安全扫描（重点🔥）

👉 检查代码问题：

* ❌ 明文密码
* ❌ 不安全权限（IAM）
* ❌ S3 公共访问风险

---

# 🧩 什么是 CK（Cloud Knowledge？）

这里的 **CK 一般指：Cloud Knowledge（云知识）**

👉 结合 CodeWhisperer：

* 根据 AWS 最佳实践生成代码
* 自动推荐架构

---

2. 登录 AWS
3. 启用 CodeWhisperer

## 方式 2：支持语言

* 输出：代码建议

```
你写一句话 → AI 帮你写代码
```

# 📌 AWS CDK + CodeWhisperer 实际使用流程

这一页 = 👉 **从安装 → 登录 → 授权 的完整流程**

---

## 🔹 Step 1：在 VS Code 连接 AWS

👉 打开 **AWS Toolkit**

你会看到：

* AWS Explorer
* CodeWhisperer 选项

👉 选择：
**CodeWhisperer（AI code suggestions）**

---

## 🔹 Step 2：创建 AWS Builder ID

页面右上图就是👇

👉 点击：
**Create AWS Builder ID**

✔ 这是一个：

* 免费账号（不用信用卡）
* 专门给开发者用

---

## 🔹 Step 3：登录（授权 VS Code）

登录后会弹出👇（右下图）：

👉 “Allow AWS Toolkit for VSCode to access your data?”

你要点：
✔ **Allow**

---

## 🔹 Step 4：授权内容（重点理解）

它会获得这些权限：

* ✅ CodeWhisperer 自动补全（核心功能）
* ✅ Code 分析（安全检查）
* ✅ IAM Identity Center 访问

👉 简单说：
👉 **让 AI 能看你代码 → 才能给建议**

---

## 🔹 Step 5：完成连接

完成后：

👉 VS Code 中就会出现：

* 自动代码建议（灰色提示）
* Tab 键补全

---

# 💡 一句话总结（考试用）

👉 CodeWhisperer 需要通过 AWS Toolkit 登录 AWS Builder ID，并授权访问后，才能在 IDE 中提供 AI 代码建议。

# 🧠 1️⃣ 一句话总结（面试直接说）

> CodeWhisperer provides inline code suggestions that can be accepted, navigated, or dismissed using simple keyboard shortcuts.

---

# ⌨️ 2️⃣ 快捷键总结（必须记住这 4 个🔥）

| 操作   | 快捷键                                  | 含义         |
| ---- | ------------------------------------ | ---------- |
| 接受代码 | `TAB`                                | 用 AI 生成的代码 |
| 切换建议 | `← / →`                              | 看不同 AI 方案  |
| 忽略建议 | `ESC` / 继续输入                         | 不用 AI      |
| 主动触发 | `Option + C (Mac)` / `Alt + C (Win)` | 让 AI 给建议   |

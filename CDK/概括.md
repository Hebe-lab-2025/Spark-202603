# 🧠 AWS CDK + Serverless + EC2 + CodeWhisperer

# 1️⃣ AWS CDK 基本流程

## AWS CDK Basic Workflow

**1. 打开项目 / Open project**

* 在 VS Code 中打开文件夹并打开终端
* Open folder in VS Code and open terminal

**2. 创建项目结构 / Create project structure**

* 创建 infra / services 文件夹
* Create `infra` and `services` folders

**3. 初始化 CDK / Initialize CDK**

* `cdk init app --language typescript`

**4. 导入模块 / Import modules**

* 导入所需 AWS 服务模块
* Import required AWS service modules

**5. 定义资源 / Define resources**

* Scope、Logical ID、Props
* Define scope, logical ID, and properties

**6. 构建（可选）/ Build (optional)**

* `npm run build`

**7. Bootstrap（一次）/ Bootstrap (one-time)**

* `cdk bootstrap`

**8. 合成模板 / Synthesize**

* `cdk synth` → 生成 CloudFormation 模板

**9. 部署 / Deploy**

* `cdk deploy`

# 2️⃣ Serverless 架构

## Serverless Architecture

使用：

* Amazon API Gateway
* AWS Lambda
* Amazon S3

### 流程 / Flow

1. API Gateway 接收请求
2. Lambda 处理逻辑
3. 从 S3 读取数据（JSON）
4. 返回结果

👉
API Gateway → Lambda → S3 → Response

---

## 关键组件 / Key Components

### 1. S3

* bucketName：存储数据
* Used to store JSON/data

---

### 2. IAM Role

* 权限控制（访问 S3 等）
* Controls permissions (e.g., S3 access)

---

### 3. Lambda

* handler：入口函数
* runtime：运行环境
* code：业务逻辑

---

### 4. API Gateway

* REST API 名称
* Resource：路径（/balanceStatus）
* Method：GET
* 连接 Lambda

---

# 3️⃣ Web Server 架构（EC2）

## Web Server Architecture

使用：

* Amazon VPC
* Amazon EC2

---

## 架构说明 / Architecture

* VPC：10.0.0.0/16
* 公有子网：运行 EC2
* 私有子网：可选

---

## VPC 配置 / VPC Config

* ipAddresses：CIDR（约 65,000 IP）
* natGateways = 0（避免费用）
* Avoid NAT Gateway to save cost

---

## Security Group

* 控制入站/出站流量
* Controls inbound/outbound traffic

示例：

* 允许 HTTP（80）
* Allow HTTP (port 80)

---

## EC2 配置 / EC2 Config

* instanceType：实例类型
* machineImage：Amazon Linux
* keyPair：登录密钥
* userData：启动脚本

---

## User Data（重点）

👉 启动时自动执行脚本
Runs script at instance startup

例如：

* 安装 Web Server
* 启动应用

---

# 4️⃣ Amazon CodeWhisperer

## AI 编程助手

### 是什么 / What is it

* AI 编码助手
* AI coding companion

---

### 功能 / Features

* 自动生成代码

* Generates code automatically

* 根据注释补全代码

* Suggests code based on comments

* 支持多语言（Python / Java / JS 等）

* Supports multiple languages

---

### 工作原理 / How it works

* 基于 LLM（大语言模型）
* Powered by Large Language Model

---

### 优势 / Benefits

* 提升开发效率

* Improves productivity

* 减少重复代码

* Reduces boilerplate code

---

### 使用步骤 / Setup Steps

1. 安装 VS Code 插件
2. 登录 AWS Builder ID
3. 启用 CodeWhisperer
4. 开始写代码自动提示

* AI boosts coding speed

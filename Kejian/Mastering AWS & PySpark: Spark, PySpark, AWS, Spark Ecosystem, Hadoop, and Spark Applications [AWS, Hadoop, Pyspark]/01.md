# 🧠 一、AWS CDK + Serverless（核心考点）

## 🔹 CDK 是什么？

👉 用代码创建 AWS 资源（Infrastructure as Code）

---

## 🔹 通用步骤（最重要！）

1. 创建项目（cdk init）
2. 导入模块（import）
3. 定义资源（Props）
4. build（可选）
5. bootstrap（一次）
6. synth（生成模板）
7. deploy（部署）

👉 一句话：
👉 **写代码 → 转 CloudFormation → 部署**

---

## 🔹 Use Case 1（Serverless 架构）

### 架构：

API Gateway → Lambda → S3

### 组件：

### 1️⃣ S3

* bucketName

---

### 2️⃣ IAM Role

* roleName
* assumedBy（谁用）
* policy：AmazonS3FullAccess

---

### 3️⃣ Lambda

* handler
* runtime
* code
* role

---

### 4️⃣ API Gateway

* handler（Lambda）
* restApiName
* resource（路径）
* method（GET）

---

👉 流程：
用户 → API → Lambda → S3 → 返回结果

---

## 🔹 Use Case 2（Web Server）

### 架构：

VPC → EC2 → Web Server

---

### 1️⃣ VPC

* CIDR：10.0.0.0/16
* natGateways: 0（省钱重点！）

---

### 2️⃣ Security Group

* 入站规则（HTTP/SSH）
* 出站规则

---

### 3️⃣ EC2

* instanceType
* AMI（Amazon Linux）
* key pair
* userData（自动安装软件）

---

👉 userData作用：
👉 启动时自动安装 Web Server（如 nginx）

# 🤖 二、CodeWhisperer（AI重点）

## 🔹 是什么？

👉 AWS 的 AI 编程助手（类似 Copilot）

---

## 🔹 功能

* 自动补全代码
* 根据注释生成代码
* 安全扫描

---

## 🔹 使用步骤（重点）

1. 安装 AWS Toolkit（VS Code）
2. 登录 AWS Builder ID（免费）
3. 点击 Allow 授权
4. 开始自动补全

---

👉 一句话：
👉 **登录 → 授权 → AI 自动写代码**

---

# ⚡ 三、Spark（大数据核心）

---

## 🔹 Spark 是什么？

👉 分布式数据处理框架（比 Hadoop 快）

特点：

* 快（内存计算）
* 分布式
* 支持 ML / Streaming

---

## 🔹 RDD（核心概念）

👉 Resilient Distributed Dataset

特点：

* 不可变
* 分布式
* 并行计算

---

## 🔹 Transformation vs Action（必考）

### Transformation（懒执行）

* map()
* flatMap()
* filter()
* distinct()

👉 不执行，只构建流程

---

### Action（触发执行）

* count()
* collect()
* saveAsTextFile()

👉 才真正运行

---

## 🔹 常见操作

### map

👉 一对一转换

### flatMap

👉 一对多（展开）

### filter

👉 过滤

---

### reduceByKey（🔥重点）

👉 按 key 聚合

例：
("apple",1) → ("apple",2)

---

## 🔹 DataFrame（更高级）

👉 类似数据库表（有列名）

---

### 常用操作：

* select
* filter
* groupBy
* orderBy

---

## 🔹 groupBy（必考）

👉 分组统计

---

## 🔹 UDF

👉 自定义函数

---

## 🔹 Cache

👉 加速计算（缓存数据）

---
